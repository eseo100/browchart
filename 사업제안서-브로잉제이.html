왜 만나뵙고 싶었나

업계 표준을 만든 분과 시스템을 만들고 싶습니다

반영구 메이크업의 운영 방식을 디지털화한 전용 SaaS, 함께 완성하실 분이 필요해요.

  • 브로잉제이 원장님은 시술뿐 아니라 강의·교육 분야에서도 업계 표준이세요.
  • 그 운영 방식과 노하우를 도구화하면, 더 많은 시술자가 같은 수준으로 매장을 운영할 수 있습니다.
  • BrowChart는 그 도구의 기반이고, 첫 파트너로 원장님을 모시고 싶습니다.
"원장님의 운영 방식 = BrowChart의 기능 설계도"
02 / 18
시장 진단

반영구 시술자가 매일 겪는 현실

  • 🔴 예약 관리의 혼란 카톡/전화/DM 다중 채널 → 누락·중복. 입금 확인은 수동, 노쇼 방지가 어려움.
  • 🔴 차트는 종이로 관리 펜·노트로 적은 차트는 분실 위험. 사진은 폰에 흩어져 있고, 시술 전·후 비교가 어려움. 리터치 타이밍을 놓침.
  • 🔴 동의서 종이 보관 시술마다 작성 + 캐비넷 보관. 분쟁 시 찾기 어렵고 분실 위험 큼.

결과: 시술 시간보다 행정에 더 시간 씀.

04 / 18
제품 소개

BrowChart — 반영구 시술자를 위한 통합 운영 SaaS

📅
예약 시스템
손님이 링크/QR로 직접 예약. 시간 충돌 자동 차단. 입금 확인 / 상태 관리.
👤
고객 차트
상담 차트(손님 입력) + 진단 차트(원장 입력) 분리. 펜 시안 (Apple Pencil 지원). 시술 전·후 사진 비교 슬라이더.
✍️
전자 동의서
카테고리별 자동 템플릿(눈썹/입술/속눈썹). 디지털 서명 + 자동 저장.
📊
운영 대시보드
월 캘린더 + 시간대별 타임테이블. 매출 분석 (메뉴별/일별). 고객 등급 자동 (VIP/단골/신규).

차별화 포인트

7대 차별 기능

기존 통합형(콜라보살롱) · 네이버 예약에 없는 반영구 전용 기능들

  • 얼굴 사진 위 디자인 시안 (Apple Pencil 지원)
  • 시술 전·후 사진 비교 슬라이더
  • 눈썹/입술/속눈썹 모양 라이브러리
  • 부위별 차트 (각 시술 특화)
  • 모듈형 동의서 + 전자서명 (카테고리 자동)
  • 리터치 자동 알림 (4~6주 후 SMS·알림톡)
  • 컬러/디자인 태그 라이브러리

협업 방식

단계별로 함께 만들어갑니다

베타 → 2차 미팅 → 추가 기능 / 가격 플랜 협의

📍 1차 미팅 (오늘)

  • BrowChart 데모 시연
  • 베타 시작 의사 결정
  • 1개월 베타 협약 체결

📍 베타 (1개월)

  • 실제 매장 사용 + 피드백
  • 주 1회 미팅
  • 즉시 수정 / 개선

📍 2차 미팅 (베타 종료 시점)

  • 추가로 필요한 기능 협의 (예: 시그니처 디자인 라이브러리, 알림톡 자동화, 인스타 공유, 고객 LTV 분석 등)
  • 매장이 진짜 필요로 하는 것부터 우선 개발
  • 추가 기능에 따라 가격 플랜 / 수익 분배 조율 가능
  • 정식 계약 체결
"무엇을 먼저 만들지·얼마에 할지" 함께 정함 → 양측 win-win

파트너십 의의

왜 브로잉제이와 함께해야 하나

🎯 정확한 페르소나

브로잉제이 원장 = BrowChart가 만들고자 하는 이상적 사용자. 베타 피드백이 제품의 깊이를 결정합니다.

🎓 수강생 → 매장 = 즉각적 유저베이스

강의 1회당 N명의 시술자가 잠재 고객. 강사가 직접 추천 → 신뢰 즉시 형성.

🤝 공동 브랜딩 효과

"브로잉제이 원장이 직접 쓰는 시스템"이라는 메시지. 마케팅 비용 절감 + 브랜드 권위.

수익 분배

브로잉제이 트레일링 커미션 30%

  • 대상: 브로잉제이 강의/네트워크 통해 가입한 매장 (추천 코드 추적)
  • 기간: 매장 가입 후 24개월
  • 수익 정의: 매장 구독료 (결제수수료 차감 후)
강의로 50매장 가입 → 19,000원 × 50 = 월 95만원
그 중 30% = 월 28.5만원
→ 연 342만원 (강의 추가 시 매장 증가 → 누적 증가)
100매장 → 월 57만원, 200매장 → 월 114만원

추가 베네핏

  • 베타 협업 1개월 별도 수당 (협의)
  • 공식 강사 권한 (BrowChart 추천 시스템 단독 진입)
  • 공동 마케팅 자료 활용권
