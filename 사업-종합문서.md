## 1.1 핵심 한 줄

> **반영구 메이크업 1인 매장 사장님을 위한 통합 운영 SaaS**

## 2.1 Executive Summary

### 회사 정보
- **회사명**: BrowChart (개인사업자, 향후 법인화 검토)
- **대표**: 장미나
- **주소**: (사업자 등록 시 결정)
- **창업일**: 2026년 4월 (개발 시작)
- **출시일**: 2026년 7월 (예정)

### 비전
**"반영구 메이크업 시술자가 시술에만 집중할 수 있도록"**

행정 업무(예약·차트·동의서·사진·정산)를 자동화·디지털화해서, 시술자가 본업에 집중할 수 있는 환경을 만든다.

### 미션
- 한국 반영구 매장 1,000개 이상 BrowChart 도입 (3년 안)
- 수강생/신규 시술자에게 표준 운영 시스템 제공
- 손님 만족도 ↑ (예약 편의성, 차트 일관성, 동의서 디지털화)

### 핵심 제안 (Value Proposition)
- 통합형(콜라보살롱) 대비 **반영구 전용 깊이**
- iPad/Apple Pencil 우선 설계
- 14일 무료 체험 + 월 19,000원 단일 플랜

### 자금 필요 여부
- **현재**: 자기 자본 (비용 거의 없음 — Vercel 무료, Supabase 무료 티어, 도메인만)
- **3개월 후**: 100만원 정도 (사업자 등록, 카카오 비즈채널 인증, 마케팅)
- **1년 후**: 외부 투자 검토 (성장 가속화 시)

## 2.2 시장 분석

### 시장 규모 (추정)

#### 한국 반영구 메이크업 시장
- 반영구 시술자 약 **5,000~10,000명** (추정)
- 시술 단가 평균 **15~25만원** (눈썹문신 기준)
- 시술자당 월 시술 평균 **30~80건**
- 시장 규모: 연 **3,000억~7,000억원** 추정

#### TAM / SAM / SOM
- **TAM (Total Addressable)**: 한국 반영구 매장 ~10,000개 × 월 19,000원 × 12 = **연 22.8억**
- **SAM (Serviceable)**: 디지털화 의지 있는 매장 ~3,000개 = **연 6.8억**
- **SOM (Serviceable Obtainable)**: 3년 내 1,000매장 = **연 2.28억** (목표)

### 시장 트렌드

1. **반영구 시장 성장**: 30~50대 여성 + 남성도 증가 (눈썹/헤어라인)
2. **디지털 전환**: 손님이 카톡/인스타로 예약 익숙
3. **콜라보살롱 유료화**: 무료 → 14일 트라이얼 + 19,000원으로 전환 → 유료 SaaS 시장 검증됨
4. **iPad 보급**: 시술자 사이에 iPad 사용 증가
5. **카톡 알림톡**: 매장이 카톡 채널로 손님 관리 익숙

### 페인 포인트 (시술자 입장)

| 페인 포인트 | 빈도 | 영향 |
|---|---|---|
| 카톡/전화/DM 다중 채널 예약 → 누락·중복 | 매일 | 매출 손실 |
| 차트 종이/노트로 관리 → 분실 | 자주 | 분쟁 위험 |
| 시술 사진 폰에 흩어짐 | 매일 | 마케팅 불가 |
| 동의서 종이 + 캐비넷 | 시술마다 | 분실, 검색 불가 |
| 리터치 타이밍 놓침 | 자주 | 재방문 손실 |
| 입금 확인 수동 | 매일 | 시간 소모 |
| 매출 정리 엑셀 / 수기 | 매월 | 분석 어려움 |

→ **결과**: 시술 시간보다 행정에 더 많은 시간 씀

### 솔루션 비교

| 솔루션 | 강점 | 약점 |
|---|---|---|
| **콜라보살롱** | 통합형, 안정적, 인지도 ↑ | 반영구 깊이 X, 일반 미용 우선 |
| **네이버 예약** | 검색 노출 강력, 무료 | 예약만, 차트/동의서 X |
| **카카오 채널** | 손님 친숙, 카톡 통합 | 차트 X, 매장이 직접 운영 |
| **종이/엑셀** | 비용 0 | 분실, 검색 불가, 백업 X |
| **MindBody (글로벌)** | 강력한 기능 | 한국어 X, 비쌈 ($30+) |
| **개별 앱** | 필요 시 사용 | 통합 안 됨, 데이터 분산 |

## 2.3 제품 / 서비스

### 핵심 기능 (현재 구현된 것)

#### 손님 영역
- 매장별 예약 페이지 (`/booking/[slug]`)
- 손님 정보 + 시술 경험 입력
- 달력 + 시간 슬롯 (요일별 영업시간, 충돌 자동 차단)
- 본인 예약 조회 (폰번호)

#### 원장 영역
- 대시보드 (실시간 통계 + 월 캘린더 + 시간대별 타임테이블)
- 시술 메뉴 CRUD (6개 카테고리)
- 예약 관리 (확정/취소/완료, 입금 확인 토글)
- 고객 차트 (상담 + 진단 분리)
  - 펜 시안 (Apple Pencil)
  - 사진 전·후 비교
  - 고객 모드 (PIN 보호)
- 전자 동의서 (카테고리별 자동 템플릿)
- 매출 관리 (기간/메뉴/일별 분석)
- 매장 설정 (정보/PIN/요일별 영업시간/휴무일)

#### SaaS 운영자 영역 (Mina)
- 모든 매장 통합 관리
- 활성도 모니터링
- 매출 분석

### 7대 차별 기능

1. **얼굴 사진 위 디자인 시안** (펜 캔버스 — Apple Pencil 지원)
2. **시술 전·후 사진 비교 슬라이더**
3. **눈썹/입술/속눈썹 모양 라이브러리** (칩 형태)
4. **부위별 차트** (각 시술 특화 항목)
5. **모듈형 동의서 + 전자서명** (메뉴 카테고리 자동 매칭)
6. **리터치 자동 알림** (35일 후 자동 등록 — SMS 연동 예정)
7. **컬러/디자인 태그 라이브러리**

### 향후 추가 기능 (베타 후 협의)

## 2.4 비즈니스 모델

### 수익 모델

```
[수익 1] 월 구독료
└─ 매장당 월 19,000원 (VAT 별도) × 매장수
   - 14일 무료 체험
   - 단일 플랜 (다단계 X)
   - 모든 기능 무제한

[수익 2 — 미래] 알림톡/SMS 발송 마진 (선택)
└─ A안: BrowChart 통합 채널
   - 매장 부담 = 알림톡 1건당 8원 (원가 6원 + 마진 2원)
   - 매장당 월 100건 = 800원 + α

[수익 3 — 미래] 결제 수수료
└─ 매장이 PG 사용 시 결제 수수료의 일부
   - 매장: 카드 2.5% 부담
   - BrowChart 마진: 0.2~0.5% (PG사와 협상)

[수익 4 — 미래] 광고 / 제휴
└─ 시술 도구·색소·바늘 등 매장 대상 광고
   - 카탈로그 / 프로모션
```

### 비용 구조

```
[고정비]
- Supabase 호스팅: 월 0원 (Free tier) → 매장 100개 넘으면 월 25달러 (3.5만원)
- Vercel 호스팅: 월 0원 (Hobby 무료) → Pro 월 20달러 (3만원)
- 도메인: 연 1.5만원
- 카카오 비즈채널 (A안 시): 월 0원 (가입 무료, 발송별 과금)
- 솔라피 마스터 계정 (A안 시): 월 0원 (가입 무료, 충전식)

[변동비]
- 알림톡 발송: 6원/건 × 매장 사용량
- SMS 발송: 9~13원/건 × 매장 사용량
- AI 추천 (구현 시): 50원/요청 × 사용량

[인건비]
- 현재: 미나님 1인 (인건비 외부 비용 X)
- 미래: 디자이너/개발자 추가 시 월 200~400만원
- 고객 지원: 매장 100개 넘으면 1명 필요

[마케팅비]
- 베타 단계: 0원 (브로잉제이 강의 활용)
- 정식 출시: 월 30~50만원 (인스타 광고)

[수수료]
- 결제 처리: 매장 구독료 × 2.5% (카드 결제 시)
- 부가세: 매장 구독료 × 10%
```

### 손익 분기점

```
고정비 = 월 7만원 (Vercel + Supabase 유료 시)
가변비 = 매장당 약 1,000원 (알림톡 발송)
순매출/매장 = 19,000원 - 1,000원 - 1,900원(VAT) - 475원(결제수수료) ≈ 15,625원

손익분기 매장수 = 70,000 / 15,625 ≈ 5매장
→ 5매장만 넘으면 흑자 (당장 도달 가능)
```

## 2.5 마케팅 + 영업 전략

### Phase 1 — 베타 (1개월, 2026년 6월)

**목표**: 브로잉제이 매장 1곳 + 첫 5명 시범 매장

**전략**:
- 브로잉제이 원장 베타 무료 사용
- 주 1회 피드백 미팅
- 즉시 반영 → 제품 완성도 ↑
- 그 사이 미나님 본인 인스타에 #BrowChart 컨텐츠 시작

**KPI**:
- 매장 1~5개 활성 사용
- 주 1회 피드백 수집
- 제품 안정성 확보 (큰 버그 0)

### Phase 2 — 강의 채널 출시 (2~6개월차)

**목표**: 첫 50매장 확보

**전략**:

```
  ※ 그 외 50매장에서 100% = 950,000원
  ※ BrowChart 합 매월 1,615,000원
```

### 매장 사장님의 여정

#### Stage 1: 인지 (Awareness)

- **행동**: 이름 / URL 메모, 인스타 팔로우
- **개선 기회**: 강력한 후기 콘텐츠, 강사 시연 영상

#### Stage 2: 관심 (Interest)
- **터치포인트**: BrowChart 웹사이트, 데모 영상
- **감정**: 흥미, "한 번 써볼까?"
- **행동**: 회원가입, 무료 체험
- **개선 기회**: 회원가입 30초 안에, 첫 매장 만들기 5분 안에

#### Stage 3: 평가 (Evaluation)
- **터치포인트**: 14일 무료 체험, 카톡 지원
- **감정**: 비교, 결정 갈등

- **개선 기회**: 온보딩 가이드, 첫 손님 예약 받는 데 도움
- **위험**: 이 단계에서 60~70% 이탈 가능 → 방지 핵심

#### Stage 4: 구매 (Purchase)
- **터치포인트**: 결제 페이지, 카톡 확인
- **감정**: 결심, 약간 부담
- **행동**: 신용카드 등록, 첫 결제
- **개선 기회**: 자동 결제 14일 후 안내, 첫 결제 후 감사 메시지

#### Stage 5: 사용 (Use)
- **터치포인트**: 매일 사용, 카톡 지원
- **감정**: 만족 / 좌절 (기능 누락 시)
- **행동**: 매일 예약 / 차트 / 사진 사용
- **개선 기회**: 정기 업데이트 노트, 새 기능 안내

#### Stage 6: 충성 (Loyalty)
- **터치포인트**: 6개월+ 사용
- **감정**: 신뢰, "이거 없으면 못 살겠다"
- **행동**: 다른 시술자에게 추천, 추가 매장 도입
- **개선 기회**: 추천 보상, 베타 신기능 우선 체험

## 3.6 수익 흐름 상세

### 단일 매장 LTV (Lifetime Value)

```
가정:
- 매장당 월 19,000원
- 평균 사용 기간 24개월
- 결제수수료 2.5% / VAT 10%

총 매출/매장 = 19,000 × 24 = 456,000원
순매출/매장 = 456,000 × (1 - 0.025 - 0.10) = 399,000원
서비스 비용/매장 = 1,000 × 24 = 24,000원 (알림톡 등)
LTV = 375,000원

CAC (Customer Acquisition Cost):
- 베타: 0 (강사 추천)
- 마케팅 단계: 1매장당 약 30,000원 (광고비)

LTV/CAC = 375,000 / 30,000 = 12.5x
→ 건강한 수치 (보통 3x 이상이면 OK)
```
