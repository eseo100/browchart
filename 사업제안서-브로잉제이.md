## SLIDE 5 — BrowChart 소개

**한 줄**: 반영구 메이크업 시술자를 위한 통합 운영 SaaS

**핵심 기능 4가지**:

📅 **예약 시스템**
- 손님이 링크/QR로 직접 예약
- 시간 충돌 자동 차단
- 입금 확인 / 상태 관리

👤 **고객 차트**
- 상담 차트 (손님 입력) + 진단 차트 (원장 입력) 분리
- 펜 시안 (Apple Pencil 지원)
- 시술 전·후 사진 + 비교 슬라이더

✍️ **전자 동의서**
- 카테고리별 자동 템플릿 (눈썹/입술/속눈썹)
- 디지털 서명 + 자동 저장

📊 **운영 대시보드**
- 월 캘린더 + 시간대별 타임테이블
- 매출 분석 (메뉴별/일별)
- 고객 등급 자동 (VIP/단골/신규)

## SLIDE 8 — 운영 흐름 (시각화)

**제목**: 시술 한 건이 시스템에서 어떻게 흘러가나

```
손님 카톡으로 링크 받음
  ↓
링크 클릭 → 매장 페이지 → 메뉴 선택
  ↓
달력에서 날짜·시간 선택 (자동 충돌 방지)
  ↓
손님 정보 입력 + 예약금 안내
  ↓ [원장에게 알림]
원장: 예약 관리에서 확인 → 확정 / 입금 체크
  ↓
시술 당일 손님 도착
  ↓
원장: 고객 차트 열기 → "고객 모드" → iPad 손님에게
  ↓
손님: 동의서 + 상담 카드 + 펜 그림 직접 작성
  ↓ [PIN 입력]
원장: 진단 차트 + 디자인 시안 그리기 → 시술
  ↓
시술 후 사진 업로드 → 시술 완료 처리
  ↓ [자동]
방문횟수 +1, 다음 리터치일 35일 후 자동 등록
  ↓
35일 후 → 자동 알림톡 → 재예약 유도
```

## SLIDE 9 — 왜 브로잉제이와 함께해야 하나

**3가지 이유**:

🎯 **정확한 페르소나**
- 브로잉제이 = 우리가 만든 SaaS의 이상적 사용자
- 베타 피드백 = 제품의 깊이를 결정

🎓 **수강생 → 매장 = 즉각적 유저베이스**
- 강의 1회당 N명의 시술자 → BrowChart 잠재 고객
- 강사가 직접 추천 = 신뢰 즉시 형성

🤝 **공동 브랜딩 효과**
- "브로잉제이 원장이 직접 쓰는 시스템"
- 마케팅 비용 절감 + 브랜드 권위

- 이 중 강의 통한 매장 = 약 50개 이상 가능

## SLIDE 12 — 수익 분배 제안

**브로잉제이 트레일링 커미션 30%**

✅ **대상**: 브로잉제이 강의/네트워크 통해 가입한 매장만 (추천 코드로 추적)

✅ **기간**: 매장 가입 후 24개월

✅ **수익 정의**: 매장 구독료 (결제수수료 차감 후)

**예시**:
- 강의로 50매장 가입 → 평균 30,000원/월 × 50 = 150만원/월
- 그 중 30% = **45만원/월** → **연 540만원**
- 강의 추가하면 누적 증가

✅ **추가 베네핏**:
- 베타 협업 1개월 별도 수당 (협의)
- 공식 강사 권한 (BrowChart 추천 시스템 단독 진입)
- 공동 마케팅 자료 활용권
